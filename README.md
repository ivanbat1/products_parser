# Installation

###### create virtual environment
```bash
python -m venv env
```
###### activate env 

on **Linux/Mac**

```bash
sourse env/bin/activate
```
on **Windows**
```bash
cd env\Scripts
activate
cd ..\..\
```

###### install packages from req.txt file

```bash
pip install -r req.txt
```

# Run

```bash
python parser.py
```
